  • 布局
    相对定位： position: relative;
    绝对定位: position: absolute; 相对父元素；
    绝对定位absolute就是相对于父元素的定位，不受父元素内其他子元素的影响；
    相对定位relative是相对于同级元素的定位， 相对自己的原来位置的移动
  • 固定 ：position: fixed;
  • z-index: 界面显示上次层次优先级；默认为0；
  • display:none; block; inline;
    块元素可以定义宽度，内联元素不可以；
  • float:浮动 clear 清除浮动；
  • overflow: 界面超出处理；
