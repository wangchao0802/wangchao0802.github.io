node 版本 v5.11.1

  • 草稿功能

草稿相当于很多博客都有的“私密文章”功能。
$ hexo new draft “new draft”
会在source/_drafts目录下生成一个new-draft.md文件。但是这个文件不被显示在页面上，链接也访问不到。也就是说如果你想把某一篇文章移除显示，又不舍得删除，可以把它移动到_drafts目录之中。

如果你希望强行预览草稿，更改配置文件：

下面这条命令可以把草稿变成文章，或者页面：
$ hexo publish [layout] <filename>

html&&css基础 流水

2017-10-16
h5
  • head ,body ,meta ,title
  • br ,p,center,
  • pre 预格式设置
  • li 列表样式 ul 无序列表 ol 有序列表
  • h*(1-6),
  • frameset 每一部分加载不同的网页，实现一个窗口加载多个页面的效果；
  • form 表单； input type=””
  • 内嵌式 ：style=”background-color:red”

  • 嵌入式

  • 引入式

  • 标签选择器；

  • class选择器； .className
  • id选择器； #id,唯一的
  • 优先级 id >class > html标签
  • 伪元素访问器；超链接的， a:link a:hover a:active a:visited
  • 颜色属性； color;
  • 字体属性； font;
  • 背景属性； background-color
  • 文本属性； text-align line-height letter-spacing; word-spacing; direction;text-transform(文本大小写)
  • 边框属性；border;
  • 列表属性；list
  • div span 为 css 标记产生，本身没任何意义；
    div 块级 span 内联元素
  • 盒子模型 width height 指的是真正的内容区域， width+border + padding+ margin = fatherViewWidth
  • 布局
    相对定位： position: relative;
    绝对定位: position: absolute; 相对父元素；
    绝对定位absolute就是相对于父元素的定位，不受父元素内其他子元素的影响；
    相对定位relative是相对于同级元素的定位， 相对自己的原来位置的移动
  • 固定 ：position: fixed;
  • z-index: 界面显示上次层次优先级；默认为0；
  • display:none; block; inline;
    块元素可以定义宽度，内联元素不可以；
  • float:浮动 clear 清除浮动；
  • overflow: 界面超出处理；
